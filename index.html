What is HTML?

HTML is a kind of language which can describe a whole webpage.

  • HTML stands forHyperText Markup Language
  • HTML is not a kind programming language，but a marketinglanguage
  • Marketing language is a set of markup tag
  • HTML uses markup tags todescribewebpage
  • HTML documentis also called web page
  • <p> defines a paragraph

    HTML Tags

  • HTML tags are some key words surrounded by <>，such as <html>
  • HTML tags also appear by pairs，such as <b> 和 </b>

    Brosers

  • Browsers(such as Google Chrome,Internet Explorer,Firefox,Safari)is used to read HTML documents ,and show it as a webpage. Browsers do not directly display the HTML tags,but HTML tags decide what to dispaly on a webpage to show to the users.
